**On this page**, you will need to introduce your worklog. **How** did you complete it? How many weeks did you take to complete the program? **What** did you do in those weeks?

Typically, and as a standard, a worklog is carried out over about 3 months (throughout the internship period) with weekly contents as follows:

**Week 1:** [Getting familiar with AWS and basic AWS services](1.1-week1/)

**Week 2:** [Account Security, Advanced VPC and AWS Vietnam Cloud Day](1.2-week2/)

**Week 3:** [Mastering EC2, LAMP/XAMPP deployment on Linux and Windows](1.3-week3/)

**Week 4:** [S3 Static Website, CloudFront CDN and Amazon RDS](1.4-week4/)

**Week 5:** [AWS Security, Serverless Research and Team Project Kickoff](1.5-week5/)

**Week 6:** [AWS Architecture Design and Project Proposal](1.6-week6/)

**Week 7:** [Midterm Review: Security and Resilient Architecture](1.7-week7/)

**Week 8:** [High Performance, Cost Optimization Review and Midterm Exam](1.8-week8/)

**Week 9:** [Building and Training Recommendation Model with AWS Personalize](1.9-week9/)

**Week 10:** [FCJ Onboarding, AWS Console and CLI Basics](1.10-week10/)

**Week 11:** [EC2 Deep Dive, SSH and Instance Creation Practice](1.11-week11/)

**Week 12:** [EC2, EBS Practice and AWS Resource Management](1.12-week12/)
