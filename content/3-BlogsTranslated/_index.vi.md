Dưới đây là danh sách các bài blog đã được dịch sang tiếng Việt:

### [Blog 1 - Tối ưu hóa vận hành SAP với CloudWatch MCP server và Amazon Q CLI – Phần 3](3.1-Blog1/)

Blog này giới thiệu cách kết hợp Amazon CloudWatch MCP Server và Amazon Q CLI để tối ưu hóa vận hành SAP. Bài viết trình bày cách sử dụng ngôn ngữ tự nhiên để tạo báo cáo sức khỏe hệ thống SAP HANA, quản lý cụm ASCS/ERS, và đơn giản hóa các tác vụ bảo trì thường kỳ. Giải pháp phù hợp với AWS Well-Architected Framework for SAP.

### [Blog 2 - Tinh gọn vận hành SAP với CloudWatch MCP Server và Amazon Q CLI – Phần 4](3.2-Blog2/)

Blog này tiếp nối phần 3, minh họa các ứng dụng nâng cao của CloudWatch MCP Server và Q CLI trong vận hành SAP. Bài viết hướng dẫn cách điều phối sự kiện bảo trì có kế hoạch, tăng tốc phân tích nguyên nhân gốc (root cause analysis), và giảm thời gian khắc phục sự cố (MTTR) cho các hệ thống SAP.

### [Blog 3 - Triển khai Okta làm nhà cung cấp danh tính tùy chỉnh cho AWS Transfer Family](3.3-Blog3/)

Blog này hướng dẫn triển khai Okta làm custom Identity Provider cho AWS Transfer Family. Bài viết trình bày cách thiết lập xác thực MFA với TOTP, cấu hình DynamoDB để quản lý người dùng, và kiểm thử kết nối SFTP với xác thực Okta.

### [Blog 4 - Bắt đầu với healthcare data lakes: Sử dụng microservices](3.4-Blog4/)
